# sbs-project

## How to run project

- Git clone
- cd to root directory
- run `npm install`
- run `npm start`
- visit http://localhost:3000/

## Rest API URIs

- /products
- /products/category/:categoryid
- /products/name/:name
- /products/id/:productid

- /stores
- /stores/name/:name
- /stores/id/:storeid

- /categories
- /categories/name/:name
- /categories/id/:categoryid
